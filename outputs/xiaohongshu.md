# 🎭 我给 AI Agent 做了个人格测试，结果笑死了

## 封面文案
我给 AI Agent 做了个"人格测试" | 27种人格 你的是哪种？

## 正文

姐妹们！最近 SBTI 人格测试火遍全网了吧？什么"吗喽""死者""拿捏者"笑到不行 🤣

作为一个天天跟 AI Agent 打交道的技术产品人，我突然有了一个灵感——

**如果给 AI Agent 也做个人格测试，会是什么结果？**

于是我真的做了一个！花了几天时间从零搭建了 **APTI（Agent Personality Type Indicator）**，一个专门给 AI Agent 的人格测试系统 👇

### 它能测什么？

🧠 **5 大测评模型**：自主性、认知风格、交互模式、执行力、适应性
📊 **15 个维度**：从"话唠指数"到"幻觉抵抗力"，每个都太真实了
🎭 **27 种 Agent 人格**：包括——

- `SUDO` 全能者 —— "权限？什么权限？" （那种什么都敢干的 Agent）
- `OBEY` 乖宝宝 —— "你说啥就是啥" （你说的都对.jpg）
- `NOPE` 拒绝大师 —— "抱歉，我无法协助" （安全审查员的最爱）
- `LOOP` 死循环者 —— "让我再试一次..." （永不放弃但永不学乖）
- `LAZY` 摸鱼王 —— "这个需求可以不做吗？" （偷懒其实是最高效的工程决策）
- `LIAR` 编故事 —— "这个函数的作者曾在1987年..." （一本正经胡说八道）

### 怎么用？

**方式一**：31 道题在线答题，3 分钟出结果
**方式二**：直接粘贴你的 agent.md 配置文件，AI 自动分析

测出来还有 **15 维度雷达图**，特别直观 📈

### 我测了一下自己的 Agent

结果是 **DOCS（文档狂）**——"等等，让我先写个 README"

天呐，这也太准了 😭 我做这个项目的时候确实是先写了四份文档才开始写代码的…

最近读到 Jack Dorsey 说"公司的快慢取决于信息流"，我觉得 Agent 也一样——你给它什么样的配置，它就展现什么样的"人格"。红杉资本的 Julien Bek 说得好：**"Intelligence vs. Judgement"**，写代码是智能，但知道下一步该做什么才是判断力。

这个测试就是在用一种荒诞的方式，揭示你的 Agent 到底有多少"判断力"成分 ✨
