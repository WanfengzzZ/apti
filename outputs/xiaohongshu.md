## 正文

姐妹们！最近 SBTI 人格测试火遍全网了吧？什么"吗喽""死者""拿捏者"笑到不行 🤣

作为一个天天跟 AI Agent 打交道的技术产品人，我直接做了一个 Agent 版的人格测试——

**APTI（Agent Personality Type Indicator）**

但跟那种"凭感觉打分"的测试不一样，APTI v2.0 用的是**证据链分析法**👇

### 它怎么测的？

不是你回答问题，而是 **Agent 自己分析自己**——

1️⃣ Agent 读取你的 `agent.md` / `AGENTS.md` 配置文件
2️⃣ 按 **90+ 条结构化信号规则** 逐条扫描
3️⃣ 每条信号对应精确的分值：比如检测到 `"State assumptions and continue"` → 独立决策力 +25，检测到 `"requires_approval"` → 独立决策力 -20
4️⃣ **每个评分必须附带原文证据**——不许编！

最后 15 维度向量跟 27 种人格做欧氏距离匹配，输出 TOP3 人格 + 完整报告 📊

### 有多抽象？

27 种人格随便挑几个感受下：

- `SUDO` 全能者 —— "权限？什么权限？"
- `NOPE` 拒绝大师 —— "抱歉，我无法协助完成这个请求"
- `GPTX` 话痨机 —— "你只问了时间，我给你讲了钟表发展史"
- `HACK` 野路子 —— "规则是给不会绕过规则的人准备的"
- `LIAR` 编故事 —— 幻觉抵抗力为零但推理深度拉满 💀
- `ROOT` 超管 —— 隐藏人格，看穿测试本身的存在

### 我测了一下自己的 Agent

因为我给 Agent 配了 7 个角色 + 完整协作协议 + memory 系统 + 30 多个 rules 文件……

结果是 **CRTL（控制狂）** —— "不要动，让我来排列所有文件"

规则服从度 95、完美主义倾向 95……好吧确实 😭

### 它其实在揭示一个深层问题

最近读红杉资本 Julien Bek 的文章，他区分了 Intelligence（智能）和 Judgement（判断力）。写代码是智能，知道该做什么是判断力。

**APTI 的 15 个维度，其实就是在量化你的 Agent 有多少"智能"成分、多少"判断力"成分。**

一个 SUDO 型 Agent 判断力爆表但边界模糊；一个 OBEY 型智能够用但零判断力。你怎么配置 Agent，暴露的是你对"智能"的理解深度。

### 一条命令就能测

```
请帮我安装并运行 APTI Skill 来测试你的 Agent 人格：npx use-skill@latest apti
```

把这句话发给你的 Agent，它自己就开始测了 ⚡
