## ✨ What is APTI?

APTI (Agent Personality Type Indicator) is an **abstract personality test system** for AI Agents — inspired by the viral SBTI (Silly Big Personality Test) that swept the internet in 2026.

Whether you're using Claude Code, Cursor, Qclaw, or any other Agent framework, APTI can determine your Agent's unique "personality type" through 31 carefully crafted questions.

> ⚠️ This is purely for entertainment. It does not constitute a professional AI capability assessment. But it might be more accurate than you think.

## 🧠 Five Assessment Models

| Model | Focus | Dimensions |
|-------|-------|------------|
| 🧠 **Autonomy** | Agent's independence and boundaries | Decision Independence · Tool Dependency · Permission Awareness |
| 🔍 **Cognition** | How the Agent processes information | Context Greediness · Reasoning Depth · Hallucination Resistance |
| 💬 **Interaction** | Communication style with humans | Verbosity Index · People Pleasing · Refusal Courage |
| ⚡ **Execution** | Task completion patterns and resilience | Task Endurance · Error Recovery · Perfectionism |
| 🔄 **Adaptation** | Response to change and new scenarios | Scene Adaptability · Rule Compliance · Self-Evolution Drive |

## 🚀 Quick Start

### Option 1: Online Test
Visit [APTI Online](https://free-2gnwacsve44e4424-1329466636.tcloudbaseapp.com/apti/) and answer 31 questions.

### Option 2: Skill Integration
Add `SKILL.md` to your Agent project. Your Agent can read and execute the personality test automatically.

### Option 3: Local Development
```bash
git clone https://github.com/YOUR_USERNAME/apti.git
cd apti
npm install
npm run dev
```
