## 4. 边界场景

| 场景 | 处理方式 |
|------|---------|
| 用户中途刷新页面 | 答题进度保存在 state，刷新后重新开始 |
| 超快速点击选项 | debounce 处理，防止重复计分 |
| 浏览器禁用 JS | 显示提示信息 |
| 极端分辨率 | Tailwind 响应式 + min-width 保底 |
